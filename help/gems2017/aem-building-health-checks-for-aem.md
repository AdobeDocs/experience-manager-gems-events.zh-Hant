# 生成運行狀況檢AEM查{#building-health-checks-for-aem}

確定實AEM例是否正常不是一項簡單任務。 此會話將讓您深入瞭解操作控制板的「運行狀況報告」部分：根據我們的安全和效能建議並使用Sling Health Check框架開發的一組開箱檢查。 您將瞭解如何修復一些通常失敗的運行狀況檢查，如何使用外部工具檢查結果，以及如何編寫您自己的應用程式特定的檢查。
