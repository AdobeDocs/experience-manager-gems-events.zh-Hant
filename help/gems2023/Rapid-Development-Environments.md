# 透過快速開發環境加速AEMas a Cloud Service開發

此網路研討會已於2023年2月22日舉行。 請檢視下面的錄製。

## 工作階段說明

為了部署變更，目前的雲端開發環境需要使用採用廣泛計畫碼安全性和品質規則的程式，稱為CI/CD管道。 對於需要快速和反複變更的情況，Adobe已引入快速開發環境（簡稱RDE）。
RDE可讓開發人員快速部署和檢閱變更，將測試經證實可在本機開發環境中運作的功能所需的時間減至最少。
