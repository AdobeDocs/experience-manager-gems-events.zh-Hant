# 在AEM中使用OSGi R7{#using-osgi-r-in-aem}

OSGi R7版本代表許多重要的新功能，並為Java開發人員認為重要且簡單易用的多項現代化技術，提供開放標準式方法。  新功能包括（其中包括）：

* 宣告式服務更新
* 新套件組合註解
* 推播資料流和承諾
* 轉換工具
* Http Whiteboard
* 設定程式和設定管理員
* Java 9支援

在這場會議中，我們將重點說明和說明OSGi R7中最有趣的新增功能，並專注於可與AEM 6.4搭配使用的功能。
