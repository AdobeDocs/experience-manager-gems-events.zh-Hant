# 在中使用OSGi R7 AEM{#using-osgi-r-in-aem}

OSGi R7版本代表了許多重要的新特性和功能，為對Java開發人員來說非常重要的一些現代價值和簡單易用的技術提供了基於開放標準的方法。  新功能包括（其中包括）:

* 聲明性服務更新
* 新建束注釋
* 推送流和承諾
* 轉換器
* HTTP白板
* 配置器和配置管理員
* Java 9支援

在本節中，我們將重點介紹和解釋OSGi R7中最有趣的增補內容，重點介紹可以與AEM6.4配合使用的內容。
