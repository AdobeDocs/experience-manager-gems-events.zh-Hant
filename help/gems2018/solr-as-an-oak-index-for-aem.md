# Solr作為Oak指數AEM{#solr-as-an-oak-index-for-aem}

全AEM文搜索功能由 [!DNL Oak Lucene] 預設情況下為索引。 但是，還AEM可以將索引和搜索卸載到 [!DNL Solr] 伺服器/群集 [!DNL Oak Solr] 索引。 在此Gem會話中，我們深入探討一些實際現實情況，其中 [!DNL Solr] 作為Oak索引，AEM在效能和索引/搜索功能方面是有意義的。
