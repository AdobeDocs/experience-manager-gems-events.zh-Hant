# 數字資產爆炸式增長 [!DNL AEM Assets]{#the-digital-asset-explosion-aem-assets}

在Adobe，我們一直在觀察數字資產在四個關鍵領域的爆炸式增長：

* 正在生成的數字資產的檔案大小
* 正在生產的數字資產數
* 我們的用戶和客戶對數字資產的期望
* 要求更大的 [!DNL AEM Assets] 部署

在本課程中，我們將瞭解數字資產爆炸，瞭解隨AEM資產爆炸的規模，並在其生命週期中遵循客戶示例 [!DNL AEM Assets] 項目。
