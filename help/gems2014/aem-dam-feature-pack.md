# 如何充分利用DAM功能包{#how-to-get-the-most-out-of-your-dam-feature-pack}

DAM FP2 for AEM 5.6.1的重點是利用Creative Projects功能為用戶提供AEM管理其資產採購的獨特方式。 該會議將介紹創意項目工作流的逐步工作，以便：

* 建立/編輯/上載鏡頭清單
* 根據快照清單上載資產
* 複查和批准上載的資產

會話還將演示任務曲面配置、自定義項目工作流模板和產品層次結構建立的能力。
