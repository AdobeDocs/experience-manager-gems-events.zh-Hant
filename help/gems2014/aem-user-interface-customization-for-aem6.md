# 用於6的用戶介面定AEM制{#user-interface-customization-for-aem}

會話將指導您瞭解如何擴展AEM6.0的用戶介面。 通過新的資源合併，可以通過覆蓋來修改每個管理控制台以更改、刪除或添加任何節點或屬性。 結合新的客戶端基礎辭彙，為產品定制開闢了新的途徑。 此外AEM，在新頁面創作中提供了廣泛的擴展點，以在分層體系結構中的定製實現中掛接。
